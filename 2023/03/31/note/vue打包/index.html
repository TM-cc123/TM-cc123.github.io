打包部署出现白屏

问题描述

 我们用vue项目在本地跑的时候一切正常，当时我们用npm run build命令打包并部署服务器后，发现Index展示的是白屏，控制台显示资源都能正确访问到

原因

出现这个问题的原因就是Vue路由模式的设置问题 > hash模式和history模式
ps:
页面空白另一种可能是文件引用路径不对，在vue.config.js里修改，之后重新打包

解决方法

导包运行本地跨域

问题

Access to script at ‘fileillDlassetslindex.c8a16327.js’ from origin ‘hul has been blocked by CORS policy. Cross origin requests are onlysupported for protocol schemes: hittp, data, chrome, chrome-extension, chrome-untrusted, https.

原因

file文件不支持import需要开启server服务

解决

1.全局安装http-server
npm install http-server -g
运行 http-server -p 8090
直接浏览器地址 localhost:8090 会自动寻找打包后的index.html文件
