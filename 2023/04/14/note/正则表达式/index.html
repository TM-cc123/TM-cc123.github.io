正则表达式

发表于2023-04-14|更新于2023-04-25|笔记
|阅读量:

修饰符 (末尾修饰;/cc/g)

  • i 修饰符:不区分大小写

  • g 全局匹配搜索

  • m 多行匹配

  • s 特殊字符圆点 . 中包含换行符

  • U 只匹配最近的一个字符;不重复匹配

    • ```js
      /^\d\w+\d$/gi

方括号

  • [abc] 查找方阔号内的任何字符
  • [^abc] 查找任何除了方括号abc的字符
  • [0-9] 查找任何从0至0的数字
  • [adgk] 查找给定集合内的字符
  • [^adgk] 查找给定集合外的字符
  • (red|blue|green) 查找任何指定的选项
    • 括号或者的意思 /(V|H)/g 全局匹配V或者H; 多个条件匹配

元字符

  • .

    • 查找单个字符,除了换行和行结束符

  • ^

    • 表示以什么开头
    • /^\d/ 字符串必须以数字开头

  • $

    • 表示以什么结尾
    • /\d$/ 字符串必须以数字结尾
    • /^\d\d$/ 表示字符串以数字开头数字结尾

  • \d

    • 查找数字

  • \D

    • 查找非数字

  • \s

    • 查找空白字符

  • \S

    • 非空白字符

  • \b

    • 匹配单词边界

  • \0

    • 查找Null字符

  • \n , \f , \r , \t , \v ;

    • 查找换行\换页\回车\制表\垂直制表符

  • \w \W

    • 匹配(非)字母\数字\下划线

量词

    • 出现0~多次

    • ```js
      /^\d*$/.test(‘’); // true 0次
      /^\d*$/.test(‘1’); // true 1次
      /^\d*$/.test(‘12’); // true 多次

  • ?

    • 表示字符串只能由0次或者1次组成 = > 有 或 无

    • ```js
      /^\d?/.test(‘abc’); // true 0个数字 捕获空字符
      /^\d?/.test(‘1abc’); // true 1个数字 捕获数字1
      /^\d?/.test(‘12abc’); // true 2个数字 捕获数字1

  • {n,}

    • 表示n~多次

    • ```js
      /^\d{3,}/.test(‘123’); // true 数字大于等于3个以上组成
      /^\d{3,}/.test(‘1234’); // true
      /^\d{3,}/.test(‘123c’); // true

  • {n,}? 或者 {n,m}?

    • 表示在捕获的时候按最少n个来捕获

    • ```js
      /\d{3,}/.exec(‘123456’) // 123456
      /\d{3,5}?/.exec(‘123456’) // 123
      /\d{3,}?/.exec(‘123456’) // 123 只按最低数n来捕获

  • 一个整体使用

    • ()

      • ```js
        let reg = /abcd{2}/; //表示d出现两次
        let reg2 = /(abcd){2}/; //表示abcd整体出现两次
        let str = ‘abcdabcd’;
        reg.test(str) //false
        reg2.test(str) //true

    • (?:)

      • ```js
        整体不捕获 ,值标志一个整体,不单独捕获
        let reg = /\d+(?:\s+\d+){2}/
        reg.exec(‘123 123 123’)

    • []

      • 一个中括号占一个字符位置

      • 表示里面任意一个都行

      • ```js
        let reg = /[abcd]/
        let reg2= /^[abcd]/
        reg.test(‘qqqawrw’) //true
        reg2.test(‘qqqawrw’) //false
        reg2.test(‘cgasda’) //true

正则预查

  • 正向预查

    • (?=) 正向肯定预查

      • 捕获一个内容时,<**后面**>必须跟着是我选择的某一个才可以

      • ```js
        let reg = /ES(?=2015|2016)/
        reg.exec(‘ES2015’) // ES
        reg.exec(‘ES2016’) // ES
        reg.exec(‘ES2017’) // null

  • 负向预查

    • (?<=)负向肯定预查

      • 捕获一个内容时,<**前面**>必须跟着是我选择的某一个才可以

      • ```js
        let reg = /(?<=2015|2016)ES/
        reg.exec(‘2015ES’) // ES
        reg.exec(‘2016ES’) // ES
        reg.exec(‘2017ES’) // null

字符串方法

  • match()

    • 找到一个或多个正则表达式的匹配。返回数组

    • ```javascript
      var str=”The rain in SPAIN stays mainly in the plain”;
      var n=str.match(/ain/g);
      // 结果 [‘ain’,’ain’,’ain’]

  • replace()

    • 替换与正则表达式匹配的子串。返回替换结果

    • ```javascript
      var str=”Visit Microsoft! Visit Microsoft!”;
      var n=str.replace(“Microsoft”,”Runoob”);
      // 结果 Visit Runoob!Visit Microsoft!

RegExp对象方法

  • compile()

    • 在 1.5 版本中已废弃。 编译正则表达式。

  • exec()

    • 检索字符串中指定的值。返回找到的值，并确定其开始匹配成功的位置。
    • /正则表达式/.exce(‘字符串’)

  • test()

    • 检索字符串中指定的值。返回 true 或 false。
    • /正则表达式/.exce(‘字符串’)

  • toString()

    • 输出正则表达式

    • ```javascript
      var patt = new RegExp(“RUNOOB”, “g”);
      var res = patt.toString();

      console.log(res) => /RUNOOB/g

  • 验证数字范围 0~255

    • ```js
      let reg = /^(1?\d{2}|2[0-4]\d|25[0-5])$/

  • 邮箱匹配

    • 名字: 6 ~ 8 位, 不能以 _ 开头, 只能由数字字母下划线组成

    • 只能是 qq 163 sina 邮箱

    • 后缀只能是 .com 或者.cn

    • ```
      let reg = /^[0-9a-zA-Z]\w{5,7}@(qq|163|sina)(.com|.cn)$/
