登录重构

发表于2022-06-29|更新于2023-04-01
|阅读量:

问题

nuxt服务端渲染无法获取正确的用户信息
服务端不能读取客户端本地存储数据

登录流程

用户点击登录 -> 请求数据 -> 返回token
储存token -> 携带token再次请求 -> 返回用户信息

解决

添加中间件 读取token

  1. 下载
    npm install cookie-universal-nuxt -S
  2. 配置 nuxt.config.js
    modules:[‘cookie-universal-nuxt’]

操作

this.$cookies.get()
this.$cookies.set()
this.$cookies.remove()

中间件读取token

  1. nuxt.config.js

    根目录新建 middleware/token.js
