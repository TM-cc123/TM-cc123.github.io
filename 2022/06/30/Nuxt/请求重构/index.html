请求重构

发表于2022-06-30|更新于2023-04-01
|阅读量:

nuxt 中的请求文件需要配置到nuxt.config.js中 -> 重构接口文件 -> 页面调用

安装nuxtjs

npx create-nuxt-app demo

配置路由

下载

npm install @nuxtjs/router -S

在nuxt.config.js配置

modules:[
‘@nuxtjs/router’
]

复制router.js到nuxt中

修改文件 命名：router.js

处理请求接口

安装axios
下载 npm install @nuxtjs/axios @nuxtjs/proxy -S
nuxt.config.js

解决跨域问题: nuxt.config.js

nuxt.config.js配置axios

新建plugins/axios.js

接口重写文件
例如 登录接口 login.js

  • 新建api文件夹存放接口
  • plugins配置 api文件路径

使用重写后的接口
异步 : asyncData()
同步 : this.$
